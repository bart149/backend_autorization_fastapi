## FastAPI бэкенд для аутентификации. Система регистрации, авторизации и управления пользователями.

### Возможности

#### - Регистрация пользователей — создание нового аккаунта
#### - JWT авторизация — безопасный вход с токенами
#### - Защищенные эндпоинты — доступ только для аутентифицированных пользователей
#### - Автоматическая документация — интерактивная Swagger UI документация
#### - Валидация данных — проверка email, паролей и имен пользователей
#### - Хеширование паролей — безопасное хранение с bcrypt
#### - SQLite база данных — легковесное хранение данных

### Технологии

#### - FastAPI
#### - SQLAlchemy
#### - Pydantic
#### - JWT
#### - bcrypt
#### - Python-JOSE
#### - Uvicorn

### Структура проекта

#### auth_app/
#### ├── main.py        # Основное приложение FastAPI и эндпоинты
#### ├── models.py      # Модели SQLAlchemy и Pydantic
#### ├── auth.py        # Логика аутентификации, JWT и хеширования
#### ├── database.py    # Настройка базы данных и подключения
#### ├── requirements.txt # Зависимости проекта
#### ├── README.md      # Документация (этот файл)
#### └── auth_app.db    # SQLite база данных (создается автоматически)

venv\Scripts\activate

source venv/bin/activate

pip install -r requirements.txt

#### Запуск приложения
python main.py

### Регистрация пользователя

POST /register

#### Описание: Создает нового пользователя с email, username и паролем.

#### Тело запроса (JSON):
{
  "email": "user@example.com",
  "username": "user1",
  "password": "secret123"
}

### Авторизация / Логин

POST /login

#### Описание: Логин пользователя. Возвращает JWT-токен для доступа к защищенным эндпоинтам.

#### Тело запроса (JSON):
{
  "username": "user1",
  "password": "secret123"
}

### Получение данных текущего пользователя

GET /users/me

### Описание: Возвращает информацию о текущем аутентифицированном пользователе. Требуется Bearer токен.

## Заголовки:
Authorization: Bearer <JWT_TOKEN>
